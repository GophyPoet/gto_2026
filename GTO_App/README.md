# Локальное приложение ГТО

Это локальное офлайн-приложение для формирования заявок ГТО из Excel-файлов школы. Оно запускается с одного файла: [index.html](./index.html).

## Быстрый запуск

1. Откройте папку `GTO_App`.
2. Запустите `index.html` в Google Chrome или Microsoft Edge.
3. На первом шаге загрузите школьный файл и файл АСУ РСО.
4. Укажите школу, директора и даты.
5. Нажмите `Проанализировать файлы`.
6. На втором шаге выберите участников.
7. На третьем шаге проверьте предупреждения и скачайте итоговый Excel.

## Как сохраняется работа

- автоматически в браузере на этом компьютере;
- через `Экспорт проекта` в JSON для переноса на флешке;
- через `Сохранить проект` в выбранную рабочую папку, если браузер поддерживает File System Access API.
