# Инструкция для учителя

## Шаг 1. Подготовка данных

Откройте `index.html`. На первом экране:

- укажите школу;
- укажите директора;
- выберите дату предоставления;
- выберите дату прохождения ГТО;
- загрузите три Excel-файла;
- нажмите `Проанализировать файлы`.

Если приложение не уверено в колонках, ниже появится блок ручного подтверждения.

## Шаг 2. Выбор участников

На втором экране:

- выбирайте класс;
- ищите ученика по ФИО;
- используйте фильтр;
- нажимайте `Добавить`.

Ниже будет показан итоговый список выбранных участников.

## Шаг 3. Проверка и выгрузка

На третьем экране:

- проверьте количество строк;
- посмотрите предупреждения;
- проверьте красные поля в предпросмотре;
- нажмите `Скачать Excel-заявку`.

## Ручное добавление

Нажмите `Добавить вручную`, заполните известные поля и сохраните. Если чего-то не хватает, приложение подставит прочерк и пометит запись для проверки.

## Как сохранить работу

- ничего не делать: браузер запомнит проект на этом компьютере;
- `Экспорт проекта`: сохранится переносимый JSON-файл;
- `Сохранить проект`: проектный JSON запишется в выбранную папку.
