# Ограничения и риски

## Ограничения браузера

Локальный HTML не может свободно записывать файлы в любую папку. Поэтому:

- итоговый Excel скачивается через браузер;
- проект сохраняется в папку только после явного разрешения;
- перенос между устройствами реализован через JSON-экспорт.

## Ограничения по Excel

`ExcelJS` хорошо сохраняет большую часть структуры, но нельзя обещать идеальное сохранение всех редких возможностей Excel.

Надежно сохраняются:

- листы;
- значения;
- большинство стилей;
- базовые объединения;
- структура шаблона.

## Ограничения исходных данных

Если в источнике нет правильного значения, приложение не может его угадать. В таких случаях оно:

- ставит `-`;
- показывает предупреждение;
- подсвечивает поле в выгрузке.
