# Требования к входным файлам

## Школьный файл

Нужен Excel, где:

- классы вынесены на отдельные листы;
- на листе есть ФИО;
- на листе есть УИН.

## Шаблон заявки

Отдельно загружать шаблон не нужно. В проект уже встроен образец `ЗаявкаШколы71_ГТО.xlsx`, по которому формируется итоговый файл.

Если позже понадобится заменить шаблон, нужен `.xlsx`, где:

- есть лист заявки;
- есть лист со ступенями;
- сохранено форматирование, которое надо унаследовать.

## Файл АСУ РСО

Нужен отчет со списком обучающихся, где есть:

- фамилия, имя, отчество;
- класс;
- дата рождения;
- пол;
- серия и номер документа;
- части адреса проживания.

## Если структура изменилась

Сначала приложение попробует найти колонки автоматически. Если это не удалось:

1. выполните анализ;
2. откройте блок ручного подтверждения;
3. выберите нужные колонки вручную;
4. сохраните проект.
